最终目的

  2. Number of arguments (or * if variable). 参数数目

  4. Number of basic blocks. 基础块数量

经常用到的类有 都是 pass 类 的派生类

  • BasicBlockPass 类。用于实现本地优化，优化通常每次针对一个基本块或指令运行
  • FunctionPass 类。用于全局优化，每次执行一个功能
  • ModulePass 类。用于执行任何非结构化的过程间优化
